Etiquetas Básicas de HTML, Atributos y Valores

En esta pagina web nos enfocaremos en explicar todas las Etiquetas con sus Atributos y valores,

para todas aquellas personas que estan empezando en el mundo de las programacion de sitios web

Nuestro proposito con este sitio web es:

Dar claridad sobre todas y cada una de las etiquetas mas usadas y de las cuales debemos conocer a la hora de utilizar html, ademas de eso conoceremos todos sus atributos como se utilizan y cuando se deben utilizar, que valores se utilizan y cuando aplicarlos; Es decir que nuestro proposito basicamente es crear una herramienta de conocimiento para que tú , que estas empezando en este mundo o que quizás ya conozcan pero quizás no conocías todas las etiquetas o que atributos y valores tienen también los conozcas y puedas utilizarlos como se debe.

1. Las etiquetas HTML, se usan para que un navegador reconozca que lo que está viendo es una página WEB todo eso que vemos cuando visitamos una página WEB vemos textos, imágenes, colores de fondo, en algunos casos elementos que se desplazan por la pantalla, en fin, todo el conjunto de elementos que hacen de dicha página un espacio multimedia e interactivo todo eso están contenido en etiquetas

2. Los atributos sirven para proporcionar información adicional sobre los elementos. Permiten establecer las características de los elementos a los que están asociados; Los atributos se colocan en la etiqueta de apertura del elemento y está integrada por dos partes: el nombre de la propiedad y el valor que se desea asignar a esa propiedad.

3.Son las propiedades que se le pueden asignar a los elementos. Por ejemplo, a HTML se le puede asignar un atributo para indicar en qué idioma está escrito el documento (lang) y podemos asignarle como valor un código de idioma ("es" para el español, "en" para el inglés...)
