Proposito del sitio

En este sitio web daremos a conocer las etiquetas básicas de HTML, también los atributos y los valores que son necesarios o son utilizados siempre en la creación de una página web.
También mostraremos el mockup el cual nos dejara una introducción de como se planeo y se organizó la creación de este sitio web.

Etiquetas básicas

Según el sitio web (Uniwebsidad, 2021) nos dice que las etiquetas utilizadas en la creación de páginas web son: a, abbr, acronym, address, applet, area, b, base, basefont, bdo, big, blockquote, body, br, button, caption, center, cite, code, col, colgroup, dd, del, dfn, dir, div, dl, dt, em, fieldset, font, form, frame, frameset, h1, h2, h3, h4, h5, h6, head, hr, html, i, iframe, img, input, ins, isindex, kbd, label, legend, li, link, map, menu, meta, noframes, noscript, object, ol, optgroup, option, p, param, pre, q, s, samp, script, select, small, span, strike, strong, style, sub, sup, table, tbody, td, textarea, tfoot, th, thead, title, tr, tt, u, ul, var. Aunque algunas de ellas ya se encuentran obsoletas, también nos dice que, aunque son muchas la etiquetas acá mencionas no es suficientes en ocasiones para una creación de una página web y que algunas etiquetas necesitan información adicional.

Valores HTML

Los valores HTML son propiedades que se les pueden asignar a los elementos de html según (Wikilibros, 2019) un ejemplo de los valores seria:
A html se le puede asignar un atributo para indicar en qué idioma está escrito el documento (lang) y podemos asignarle como valor un código de idioma ("es" para el español, "en" para el inglés...) ... O sea: ... Por supuesto un elemento puede tener varios atributos: ... Los atributos deben estar en la etiqueta inicial, nunca en la final. El orden de los atributos dentro de la etiqueta es indiferente. Deben separarse con un espacio en blanco del nombre del elemento y de otros atributos. Atributo y valor deben estar unidos por el signo: = Los valores deben estar encerrados con comillas, dobles o simples. Esto no es totalmente cierto, pero actúa como si lo fuera.
